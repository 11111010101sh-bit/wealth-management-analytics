# Wealth Management Analytics Project

##  Problem Statement
To analyze client portfolio and transaction data to support Wealth Management decision-making using data analytics.

##  Tools Used
- SQL
- Python 
- Excel

##  Relevant Business Questions
- Which clients contribute the most to Assets Under Management (AUM)?
- How do portfolio returns vary by risk profile?
- Which clients show low engagement despite high value?
- What investment types dominate client portfolios?

##  Insights Generated
- Identified top clients contributing to overall AUM
- Calculated portfolio returns to assess performance
- Highlighted engagement gaps for potential follow-up
